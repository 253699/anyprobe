# Any Probe
[中文版本](README.md)

Build a lightweight basic monitoring system and display it with an elegant visual interface.
- Official Website: https://anyprobe.ipv4.website  
- GitHub: https://github.com/253699/anyprobe  
---
## Recommended Systems
Alpine 3.20, CentOS Stream 9, Debian 12, Ubuntu 24.04, Windows 10
---
## Motivation
Minimal performance impact with elegant display of basic server information.  
In non-rigorous testing on Alpine 3.20, the probe uses only 3MB of memory with extremely low CPU usage.
