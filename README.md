# Any探针
[English Version](README_EN.md)
构建一个轻量的基础监控，并优雅地展示可视化页面。
- 官网：https://anyprobe.ipv4.website  
- GitHub：https://github.com/253699/anyprobe  
---
## 推荐系统
Alpine 3.20、CentOS Stream 9、Debian 12、Ubuntu 24.04、Windows 10
---
## 初衷
几乎不占用多少性能，优雅地显示服务器基础信息。  
不严谨的测试在 Alpine 3.20 下，探针内存仅占 3MB，CPU 占用极低。
